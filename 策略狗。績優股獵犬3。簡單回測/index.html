策略狗。績優股獵犬3。簡單回測

2017-09-23
歷史回測, 策略狗, 自由現金報酬率

今天來做個間單的回測，為策略狗文章作一個總結：

我們選股條件：

  1. “自由現金報酬率”最高的前30%
  2. 本益比 < 25
  3. 股價淨值比 < 1.5

每格N天就重新選一次手中的股票，比較跟策略狗不一樣的是，回測不只限於20支，只要滿足條件，就會買入，例如當手上要換成30張股票時，會將自己的所有現金平均分配到此30張股票，不計算手續費。

幫大家測完的結果，雖然我寫的方法跟策略狗有點不一樣，但是用非常雷同的方法，結論是：真的每年再換一次股就好了！
